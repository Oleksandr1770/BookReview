About

This website is meant for teaching the content from the books “Don't Make Me Think, Revisited: A Common Sense Approach to Web Usability” and “Rocket surgery made easy: the do-it-yourself guide to finding and fixing usability problems.” The creation of this website is based on a number of requirements:

  1. Create a simple website that is hosted online (submit the link in a text file to D2L Dropbox) and has no pop ups
  2. Implement a mechanism for navigating your website that takes advantage of the fact that web users "browse" content much like walking into a clothing store, clicking on the first target of opportunity even if it doesn't 100% match the user's ultimate goal.
  3. Implement anchor points (images located intelligently) to guide the user’s thoughts through a storyline of the books.
  4. Ensure that your web design minimizes the cognitive load necessary to browse your content
  5. Implement action words in your text on the left-hand side of the screen, taking advantage of the fact that users do not read sequentially through bodies of text
  6. Ensure that the user does not pose questions when navigating your site
  7. Implement a UI design that takes advantage of the following UI design guideline: Don't squander the user's patience
  8. Implement a singular strong focal point in the main page of your site
  9. Make navigational elements obvious
  10. Make use of effective writing
  11. Include plenty of white space in your design
  12. Communicate using visible language
  13. Use at least one common web convention (i.e. search bar top right corner)
  14. Website is visually appealing, uses proper grammar and spelling, and is a pleasure to navigate
